# Interactive Registration Form - React.js Project

A clean and well-structured user registration interface built using React.js, focusing on state management, controlled inputs, and layout design during my front-end development internship.

## 📅 Project Details
- **Role:** Front-End Development Intern
- **Timeline:** July 7, 2026 (Internship Task)
- **Status:** Completed

## 🛠️ Technologies & Tools Used
- **React.js:** Functional components and state management hooks.
- **JavaScript (ES6+):** Handling form data events and conditional structures.
- **CSS3:** Custom layout design optimized for desktop and laptop monitor layouts.
- **Development Environment:** Visual Studio Code (VS Code)

## 📌 Form Features & Inputs
The form captures core user details with controlled state handling:
- **First Name & Last Name:** Structured text inputs.
- **Date of Birth:** Optimized calendar/date input field.
- **Email Address:** Standard validated text input field.
- **Gender Selection:** Dynamic selection controls.
